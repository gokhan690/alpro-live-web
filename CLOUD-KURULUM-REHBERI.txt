ALPRO V6.0 - SUPABASE CLOUD + HOSTING KURULUMU

SUPABASE TEK BASINA YETMEZ:
Supabase veritabanidir. Uygulamanin internette acilmasi icin server.js bir hosting uzerinde calismalidir.

ONERILEN KOLAY YOL: RENDER.COM

1) Supabase tarafinda:
   - SUPABASE-SETUP.sql dosyasini SQL Editor'de calistir.
   - Project Settings > API bolumunden:
     SUPABASE_URL
     service_role key
     bilgilerini al.

2) Render tarafinda:
   - render.com hesabina gir.
   - New > Web Service sec.
   - Bu klasoru GitHub'a yukleyip bagla veya manuel deploy yontemini kullan.
   - Build Command: npm install
   - Start Command: npm start

3) Render Environment Variables:
   SUPABASE_URL=https://xxxx.supabase.co
   SUPABASE_SERVICE_KEY=service_role_key
   PUBLIC_URL=https://senin-render-linkin.onrender.com

4) Deploy bitince ekip linkten girer:
   https://senin-render-linkin.onrender.com

GUVENLIK:
- SUPABASE_SERVICE_KEY asla index.html icine yazilmaz.
- Key sadece Render Environment Variables veya local .env icinde kalir.
- Ekip sadece web linkini kullanir.
